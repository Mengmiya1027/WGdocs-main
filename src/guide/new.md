## 自制音频控件 <badge type="warning" text="Dev2.0,Preview15" /> <badge type="tip" text="1" />

使用方法：

::: code-group
```[HTML]
<MusicPlayer
  title="[歌名]"
  author="[歌手]"
  tlink="[封面链接]"
  flink="[音频链接]"
/>
```
```[示例]
<MusicPlayer
  title="浮游"
  author="周深"
  tlink="../other-pictures/songs/浮游.png"
  flink="../songs/%E5%91%A8%E6%B7%B1%20-%20%E6%B5%AE%E6%B8%B8.flac"
/>
```
:::
