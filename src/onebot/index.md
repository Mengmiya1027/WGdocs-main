# NapCat-OneBot 实现

NapCat 尽量遵守了大部分 OneBot 规范。在遇到无法实现或需要扩展的场景时，会进行差异化实现。

请注意，OneBot 是一个已经脱轨且停滞的协议规范，对框架本身没有实质性约束。
